
websocket 演示项目。

0.需预先安装 nodejs 和 yarn 环境

安装 & 运行：

```
git clone https://github.com/ignaciozhuzhu/websocket_demo.git
cd websocket_demo
yarn

客户端：
yarn dev

服务端（nodejs）：
再打开另一个powershell
cd websocket_demo/ch1
node server
```

打开 http://localhost:8091/ch1/client.html 即可看到效果。
